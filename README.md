# Football Prediction Model

## Goal

Predict football match outcomes using statistical and machine learning models.

## Data Sources

- football-data.org API
- Oddsportal
- Betexplorer

## Tools

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost

## Structure

```
├── data/
│   ├── raw/          # Raw data from APIs
│   └── processed/    # Cleaned data
├── models/           # Trained ML models
├── experiments/      # Model experiments
├── scripts/          # Data collection scripts
│   ├── get_matches.ps1
│   ├── get_standings.ps1
│   └── get_hkjc.ps1
├── results/          # Prediction results
└── docs/            # Documentation
```

# football-prediction

足球比赛预测系统 | Football Match Prediction System

## 模型版本

| 版本 | 准确率 | 说明 |
|------|--------|------|
| v6.1 (最新) | **62.2%** | 规则模型 + LOO验证 |
| v5 | 62.2% | 网格搜索权重 |
| 市场基准 | 54.1% | 纯赔率概率 |

## v6.1 改进点

1. **冷门检测**：强队赔率异常上升 → 反买客胜
2. **资金情绪**：活跃资金(>5次变化) + 赔率方向 → 顺势
3. **平局保护**：高平局概率 + 活跃资金 → 补平局
4. **价值 Kelly**：计算价值投注机会
5. **LOO 验证**：留一法交叉验证，泛化能力稳健

## 分联赛效果

- 强队主场(ih<2.0): **73%** vs 市场67%
- 客队热门(ia<ih): **62%** vs 市场46%
- 中性赔率: 44%（最难预测）
